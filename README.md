YukkuriMovieMaker4（YMM4）上で動作する、**映像をボロノイ分割による多面へ分割し、面ごとの屈折・色分散・反射・照明を Direct2D カスタムピクセルシェーダーで合成する映像エフェクトプラグイン**です。
面ごとの法線をシードから決定論的に求め、波長ごとに異なる屈折で色分散を生じさせ、Fresnel 反射・鏡面ハイライト・面境界の反射線を方向性のある光源で合成します。
面の形を崩す不規則さ、面の傾きを時間で動かすゆらぎ、面ごとに明滅するきらめきのパラメーターを備えます。
数値パラメータはアニメーションに対応しています。

## 概要

本プラグインは YMM4 の「映像エフェクト」として動作し、エフェクトの種類一覧では「多面体ガラス」として表示されます。

Direct2D カスタムピクセルシェーダーがフレームごとに画面をボロノイ分割し、セルごとのハッシュから各面の法線を求めます。面法線に基づいて背景映像を屈折させ、赤・緑・青を異なる波長の屈折率でサンプリングすることで色分散を生成します。さらに Schlick 近似による Fresnel 反射・鏡面ハイライト・面境界の反射線を方向性のある光源で合成し、ソース画像のアルファでマスクした多面体ガラスを生成します。

このエフェクトは AviUtl 向けの EXO 出力（`.exo`）には対応していません。

## インストール方法

1. [Releases](https://github.com/routersys/YMM4-FacetedGlass/releases/latest) ページから最新のプラグインファイル（`.ymme`）をダウンロードしてください。
2. YMM4 が起動していないことを確認し、ダウンロードしたファイルを実行（ダブルクリック）してインストールします。
3. YMM4 を起動し、タイムライン上のアイテムに映像エフェクトを追加します。
4. 映像エフェクトの種類として **「多面体ガラス」** を選択してください。

## 主な機能

### 1. ボロノイ分割による多面生成

画面をボロノイ分割し、各セルを一つのガラス面として扱います。面サイズでセルの平均的な大きさを、分割角度で全体の向きを指定します。不規則さでセル中心の揺らぎを与え、規則的な六角形から有機的な破片まで面の形を変化させます。面の傾きはシードとゆらぎから決まり、凹凸でその強さを調整します。

### 2. 色分散付き屈折

各面の法線に従って背景の映像をずらして屈折を表現します。屈折率を基準として、赤・緑・青を異なる波長の屈折率でサンプリングし、分散の値に応じて色のにじみを生じさせます。

| チャンネル | サンプリング波長 |
|---|---|
| 赤 | 610 nm |
| 緑 | 550 nm |
| 青 | 460 nm |

### 3. Fresnel反射と面境界

視線方向と面法線から Schlick 近似による Fresnel 反射を求め、方向性のある光源による鏡面ハイライトと、面境界に沿った反射線を加えます。反射の強さは反射パラメータで、境界線の幅は面境界パラメータで調整します。きらめきは面ごとに鋭く明滅する閃光を加え、ゆらぎと連動して位相が変わります。

### 4. アニメーション対応パラメータ

強さ・面サイズ・不規則さ・凹凸・分割角度・ゆらぎ・屈折・屈折率・分散・反射・きらめき・面境界・光源角度・光源高度の 14 項目はアニメーション対応であり、キーフレームを設定することで時間に応じて値を変化させることができます。

### 基本グループ

| パラメータ名 | 型 | デフォルト | スライダー表示範囲 | アニメーション | 説明 |

### 面構造グループ

| パラメータ名 | 型 | デフォルト | スライダー表示範囲 | アニメーション | 説明 |

| 不規則さ | 数値 | 65% | 0 〜 100% | ✔ | 0% で正六角形、100% で有機的な破片状の面になります。 |

## 制限事項

- AviUtl 向けの EXO 出力（`.exo`）には対応していません。
- 透明部分（アルファが 0）には効果が適用されません。

## 注意事項

- **AviUtl 非対応**: 本プラグインは AviUtl 向けの EXO 出力（`.exo`）に対応していません。
- **シードと再現性**: シードが同じであれば、面の配置と傾きは常に同じ結果になります。
- **本プラグインを使用する前に、YMM4 プロジェクトファイルのバックアップを作成することを推奨します。**
